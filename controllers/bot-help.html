Bot Help Hello and welcome to the rudimentary bot help file. Right now this bot is simple, accepting all commands in the following format "(command) (options)". There are currently two commands available for the bot Report and Forecast Using the report will generate a HTML-Formatted table of the last stashed Wx data. Forecast will query the remote OpenWeather API and return a forecast for your options. Please see the list below for available commands and options. Both commands and options are case insensitive. •Report ------ All ------ Temperature •Forecast ------ Summary ------ Temps ------ Daylight
